# 图形化工具：Navicat

如果你已经开启服务器安全组3306端口，并设置完成 MySQL 远程连接，可以参考如下的步骤使用 Navicat：

  1. 打开Navicat Premium，点击顶部菜单栏， 【文件】>【新建连接】>【MySQL】

  2. 自定义连接名，将localhost更改为数据库所在的主机公网IP地址

  3. 输入账号和密码（不知道账号密码？）

  4. 点击“测试连接”，系统提示：连接成功 ! 即表示成功连接
